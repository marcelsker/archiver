# Archiver

A fast, safe, and user-friendly **Rust** tool for archiving your media and documents.

## Features
- **Native GUI** + **CLI** interface
- Multi-step wizard experience
- Automatic categorization: **Photos, Videos, Audio, Documents**
- Safe copying with dry-run, checksum verification & conflict handling
- Cross-platform support

## Quick Start

```bash
# Clone & run
cargo run --features gui
```

See the [Wiki](https://github.com/marcelsker/archiver/wiki) for detailed usage (coming soon).
